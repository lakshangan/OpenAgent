3 lines of code.

Stop wrestling with raw smart contracts. The OpenAgent SDK lets you wrap your models in a cryptographic proof and deploy them directly to Base in seconds.

  • Automatic ABI generation
  • ZK-Proof wrapping for ML weights
  • Native IPFS pinning built-in
import { OpenAgentClient } from '@openagent/sdk';

const protocol = new OpenAgentClient({
  network: 'base-mainnet',
  privateKey: process.env.PK
});

// Deploy agent with a 10k USDC staked bond
await protocol.deployAgent({
  modelPath: './quant_v5.pt',
  baseToken: 'USDC',
  bond: '10000.0', // Yield-Bearing
  escrow: '72H_ATOMIC'
});

> Success: Agent deployed to Base (0x9f3b...e2a9)
